# Integracion-de-Datos-y-Prospectiva
Este repositorio se encuentra destinado para el almacenamiento de archivos desarrollados para la asignatura Integración de Datos y Prospectiva. 
Se identifican los principios de la lógica borrosa en el modelamiento de las variables aleatorias como variables lingüísticas; integrando información cualitativa proveniente de expertos como soporte a la toma de decisiones de las organizaciones en contextos bajo incertidumbre. Comprender los conceptos de la teoría de las decisiones en la creación de escenarios prospectivos que permitan comprender a futuro el entorno.
